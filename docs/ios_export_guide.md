# iOS Export Guide

1. Install Godot 4.x with iOS export templates.
2. Open `project.godot`.
3. Confirm portrait orientation in Project Settings.
4. Create an iOS export preset.
5. Set bundle identifier, version, display name, and icons.
6. Export an Xcode project.
7. Open the exported project in Xcode.
8. Select the signing team and provisioning profile.
9. Build on a real iPhone.
10. Test airplane mode, background/resume, notch safe areas, and home indicator spacing.

The MVP does not require network, login, analytics, ads, location, camera, microphone, contacts, or push notification capabilities.
