# Known Limitations

- The map is an 8-territory first playable slice, not the full 15-20 territory MVP map.
- Audio is generated placeholder tone feedback.
- Territory art is simple polygon data.
- AI personalities are deterministic and intentionally simple.
- Save migration currently supports only version `1`.
- No iOS-specific haptic tuning has been done on device yet.
- There is no monetization, online mode, account system, analytics, or cloud save.
